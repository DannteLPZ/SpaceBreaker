This is a Unity 2022.3.49f1 project for a technical test.

This project aims to recreate an "Arkanoid-type" game.

Supports Keyboard and Gamepad.

Controls:

Keyboard:
  - Movement: WASD or arrow keys.
  - Submit: Enter
  - Launch: Spacebar
  - Pause: Esc
    
Gamepad:
  - Movement: Left stick
  - Submit: Enter
  - Launch: Spacebar
  - Pause: Start
